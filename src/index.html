Docker is an open-source platform that enables developers to automate the deployment, scaling, and management of applications within lightweight, portable containers. Containers package an application and its dependencies, ensuring consistent performance across different environments 1 . Docker uses operating system-level virtualization to deliver software in packages called containers, which are isolated from one another and bundle their own software, libraries, and configuration files
